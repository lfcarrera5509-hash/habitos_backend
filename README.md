# Proyecto Hábitos

Aplicación full-stack desarrollada con:

- backend/: Express + MongoDB Atlas + Mongoose
- frontend/: Next.js + Redux + Tailwind CSS

## Semana 1
- Backend con CRUD de hábitos

## Semana 2
- Frontend en Next.js
- Redux integrado
- GET de hábitos desde backend

## Semana 3
- Tailwind CSS integrado
- Lista dinámica de hábitos usando Redux
- Barra de progreso visual
- Botón "Done" visual

## Cómo ejecutar backend
cd backend
npm install
npm run dev

## Cómo ejecutar frontend
cd frontend
npm install
npm run dev
